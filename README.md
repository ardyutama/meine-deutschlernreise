## Start Here

1. Open [a1/00-getting-started.md](a1/00-getting-started.md).
2. Follow the full A1 plan in [a1/README.md](a1/README.md).
3. Use the supporting files in [a1/resources/](a1/resources).
4. Use [AGENTS.md](AGENTS.md) to generate the next level into its own folder.

## Generation Rule

When adding a new level, keep it self-contained:

- Main plan in `<level>/README.md`
- Setup notes in `<level>/00-getting-started.md`
- Research and guides in `<level>/resources/`
- Design/specs in `<level>/docs/`

The A1 folder is the reference implementation for the rest of the journey.
