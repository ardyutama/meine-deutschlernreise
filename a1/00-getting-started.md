# Getting Started — Setup Guide

Complete everything on this page **before** starting Week 1. Takes about 30 minutes.

---

## 1. Install Anki (Spaced Repetition)

Anki is your most important tool. You'll use it every single day.

### Download
- **Desktop:** https://apps.ankiweb.net/ (Windows/Mac/Linux — free)
- **Android:** AnkiDroid (free) from Google Play
- **iOS:** AnkiMobile ($24.99) OR use AnkiWeb in browser (free)
- **Web:** https://ankiweb.net/ (sync across devices — free account)

### Configure Settings
1. Open Anki → Tools → Preferences
2. Set these values:
   - New cards/day: **15**
   - Maximum reviews/day: **100**
   - Learning steps: `1m 10m 1d`
   - Graduating interval: `1 day`
   - Easy interval: `4 days`

### Download Starter Decks
Go to https://ankiweb.net/shared/decks and search for:
- [ ] "German A1 Core Vocabulary" — download and import
- [ ] "German Gender: Der Die Das" — download and import

### Card Creation Rules
When making your own cards from Netzwerk Neu:
- **Nouns:** Always include the article! `der Tisch` not just `Tisch`
- **Front of card:** English word/phrase
- **Back of card:** German + article + example sentence
- **Verbs:** Include ich/du/er forms: `machen → ich mache, du machst, er macht`
- **Color code genders** (in card styling):
  - 🔵 Blue = der (masculine)
  - 🔴 Red = die (feminine)  
  - 🟢 Green = das (neuter)

## 6. Mindset

Before you start, know these truths:

1. **You will feel stupid.** This is normal. Every polyglot felt this way at the start.
2. **Consistency beats intensity.** 30 min every day > 4 hours on Saturday.
3. **Mistakes are learning.** Every error is your brain building new connections.
4. **You don't need perfection.** A1 is about basic communication, not eloquence.
5. **Fun prevents burnout.** If it's not enjoyable, change the resource, not the goal.
6. **Comparison is poison.** Your only benchmark is yesterday-you.
